Introduction

This website contains all the code used for the analysis.

Simulation Pipeline

This pipline is written in snakemake. To run the entire simulation pipeline you simply need to swap in the correct snakefile.

  • code/Snakefiles/4PopSplit/snakefile is the correct file to run the 4 population toy model from Figure 1.
  • code/Snakefiles/SimpleGrid/snakefile runs the stepping stone grid simulations and has options to simulate the confounders and do the polygenic adaptation tests from Figure 2.

The detailed methods and results for each simulation can be found here.

The pages below explain the 5 major steps of the pipeline:

  1. Simulate Genotypes
  2. Simulate Phenotypes
  3. Project Test Vector
  4. Run GWAS and Ascertain SNPs
  5. Polygenic Adaptation Test
